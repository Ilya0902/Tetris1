Минималистичная консольная реализация классического Tetris, построенная на логике координат фигур (без матриц).
Проект фокусируется на архитектуре и механике игры, а не на графике или управлении.


О проекте

Kогическая модель тетриса:
 • Игровое поле 10×20
 • Случайная генерация фигур
 • Повороты
 • Проверка столкновений
 • Очистка заполненных линий
 • Автоматическое падение фигур (tick)

❗ В текущей версии:
 • Нет подсчёта очков
 • Нет ускоренного падения
 • Нет управления с клавиатуры
 • Движение влево/вправо реализовано логически, но не привязано к вводу
